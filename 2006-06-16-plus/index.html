A 雙永國小附設幼兒園

研發主題：Chef（廚房裡的英雄）

參考繪本：《廚房裡的英雄》（大衛·昂格 / 小魯文化）


原始參考桌遊：大富翁、美味顛翻天


📋 構思設計重點整理

  • 設計背景：以繪本《廚房裡的英雄》為基礎，結合樂高自製人偶、平底鍋拋卡牌等實物配件。
  • 核心機制：設計了「甩鍋大師」（動作敏捷、卡牌配對）與「完美酷客」（合作跑丁、能力差異、情境解決）兩種不同深度的玩法。
  • 融入概念：傳遞「職業與家務不受性別限制」的性平概念，並透過情境卡融入 SEL 學習（如衝動控制、情緒調節、團隊領導力）。

💖 作品特色

一、將「廚房」作為遊戲場景，非常貼近幼兒的生活經驗


二、 運用「特質旗幟」的設計（力量、細心、溫柔、堅韌等）


三、作為角色能力，讓孩子在合作中體驗到「每個人都有獨特價值與貢獻」


四、 拋接食材的「甩鍋大師」動態玩法符合幼兒的好動天性，兼具體感操作與認知配對，兼顧動靜態發展

🚀 思考與發展方向

  1. 玩法二與玩法三的整合優化：
    如果分開玩時，機制單薄，可以把「甩鍋」作為遊戲板塊中的一個隨機挑戰（例如踩到「甩鍋點」時，需要在一分鐘內成功甩起某個食材卡才可獲得該食材），把甩鍋動作融入主要的合作送餐遊戲中

  2. 性平情境卡的深度引導：避免問題過於單一

    （例如「中央廚房需要大量豆泥只有男生能做，請表達想法」可能容易引導出刻板印象）

    可以將性平問題轉化為情境選擇，例如：「客人說：『這家餐廳的主廚一定是男生！』，我們該怎麼用我們的特質去回應他？」或「小男生想穿粉紅色圍裙，但被別人笑，我們該如何支持他？」，引導幼兒具體表達想法。

  3. 能量機制的平衡：「體力耗盡」會讓遊戲體驗中斷，可以設計「完成送餐-->恢復能量」，讓孩子持續挑戰的動力

🎲 炒麵大師 (Wok on Fire)

這是一套極具體感與歡樂效果的卡牌桌遊。玩家必須用實體塑膠鏟子在桌面大鍋中翻動食材卡牌，並按照食譜配對得分。

💡 推薦理由：這與「甩鍋大師」的體感玩法極為契合，用鏟子翻牌非常吸引幼兒，有出中文版本且極易在各大桌遊店或網路上買到。

B 福德國小附設幼兒園

研發主題：珍古德與黑猩猩的故事

參考繪本：《黑猩猩的好朋友》（珍·古德傳記繪本）

原始參考桌遊：推倒提基

📋 構思設計重點整理

  • 設計背景：以《黑猩猩的好朋友》為起點，引導幼兒探索女性在科學、環境保護與領導角色上的表現。
  • 核心機制：參考經典桌遊《推倒提基》的機制，設計用功能卡上下移動「彩虹磚頭」的版塊機制。
  • 融入概念：引導孩子理解「女性也能成為領導角色」，並在回答問題時，練習以清晰的口語表達想法。

💖  作品特色

一、選擇珍古德作為主題，切入「女性也能成為優秀科學家與領導者」


二、把 《推倒提基》玩法簡化為「彩虹磚頭」的上下移動，能讓幼兒在空間認知（上下、順序）得到鍛鍊。 


 三、配合提問卡鼓勵，可以鼓勵口語表達


四、將彩虹色塊結合科學探索，很有挑戰性

🚀 進一步發展的方向

  1. 目前構思表缺少了玩法二（性平概念）和玩法三（SEL）

  2. 建議可以將「珍古德與黑猩猩的野外觀察」或「森林保育」融入

    例如：

    玩法二（性平）：設計「觀察員任務卡」，卡牌內容是不同性別的人從事野外調查、動植物照護等工作，破除職業刻板印象

    玩法三（SEL）：融入動物保育與同理心，設計「黑猩猩生病/難過時」的情境提問

  3. 口語提問與功能卡機制的結合：建議將「提問卡」與「功能卡」做聯動。

    例如：當玩家抽到提問卡並勇敢說出自己的想法後，才能獲得「功能卡」來移動彩虹磚頭。

    這能將「口語表達」與「遊戲推進」緊密結合，讓孩子更有動機表達。

  4. 簡化計分機制：隱藏任務計分對幼兒來可能過於複雜

    建議改為合作模式：全員共同合作，在特定回合內將彩虹磚頭推到頂端，中途需回答問題以清除阻礙障礙。

研發主題：保持好奇心：從霍金看見宇宙與性別的無限可能

參考繪本：《保持好奇心！史蒂芬．霍金的生命故事》、《以馬內利，夢想起飛了》、《我還有一隻腳》

原始參考桌遊：搞笑瑜珈、鼠來堡、迷宮滾球

📋 構思設計重點整理

  • 設計背景：結合霍金與以馬內利等身心障礙者勇敢逐夢的故事，將「身心限制」體驗轉化為遊戲的體感核心機制。
  • 核心機制：4人合作拉繩控制太空船（中空圓環），運送彈珠（生命值）在方格板上移動，需避開黑洞（鏤空洞口）抵達終點。
  • 融入概念：引入「身體限制卡」、「霍金挑戰卡」（限制控繩部位，如改用手肘或下巴），讓幼兒感同身受身體限制者的不便。並設計了犯錯時全員齊跳「鼓勵之舞」的溫暖支持機制。

💖 給予支持與鼓勵

幸安附幼的設計令人驚艷！這是一套完成度極高、且非常具有創意的「共感學具」。 透過「拉繩太空船」與「霍金挑戰卡」的設計，玩家不只是聽故事，而是用自己的身體去體驗「身體受限時，如何與他人合作達成目標」，這對培養幼兒對身心障礙者的同理心與多元包容具有無可比擬的效果！ 而「鼓勵之舞」更是神來之筆，將犯錯與失敗轉化為正面且歡樂的集體社交支持，徹底實踐了 SEL 心理韌性與團隊合作的真諦！

🚀 進一步發展的方向

  1. 簡化規則與配件以利幼兒專注：玩法三的元素非常多（寶石、四種功能卡、醫療站/觀測站的開關、鼓勵之舞、推板塊）。對幼兒來說，要同時控制平衡、注意卡牌限制、還要記憶複雜的地圖功能會有些手忙腳亂。建議將「推動板塊」和「寶石收集」進行適度簡化。
  2. 物理配件的友善度調整：繩子控制中空環的穩定性對幼兒是高難度挑戰。建議太空船載具的圓環可設計有深淺不同的凹槽：初階版凹槽較深，彈珠不易掉落；高階版（霍金挑戰）凹槽較淺，增加難度。這能適應小班到大班不同年齡的發展水準。
  3. 遊戲盒尺寸與結構強度：拉繩在遊玩時會對紙盒產生較大拉力，可採用市售「滑溜騎士 (Slide Quest)」的塑膠底盤與槓桿結構作為開發參考，確保學具耐用度。

🎲 滑溜騎士 (Slide Quest)

玩家合作利用紙盒四邊的黃色槓桿來傾斜底盤，控制底座上有滾珠的騎士移動，避開陷阱並推動敵人到達終點。

💡 推薦理由：這與幸安的「拉繩控球」概念完全一致，是現成且大受好評的物理合作遊戲，有中文版且極易買到，是完美的結構參考。

參考繪本：《為冥王星命名的女孩-威妮夏 伯尼的故事》、《打破框架的天文學家-瑪麗亞 米契爾的故事》

原始參考桌遊：小丑馬戲團、大頭娃娃

📋 構思設計重點整理

  • 設計背景：將宇宙探索、星球命名與女天文學家的堅毅精神融入遊戲中，打破「女生不適合學科學」的偏見。
  • 核心機制：設計「雙面遊戲板」（A面合作、B面競賽）。玩法一為星球命名與記憶搶答；玩法二與三結合了擲骰移動與「將星星配件疊在棋子托盤上」的物理平衡考驗。
  • 融入概念：認識女性天文學家角色、鍛鍊口語表達（命名與分享原因）與傾聽（回答他人星球的名字及理由）。

💖 給予支持與鼓勵

東門附幼的設計非常精緻且有深度！以宇宙、星星為意象，成功地將科學史上的女力故事轉化為幼兒遊戲。 最出彩的是「托盤疊星星」的體感機制，把抽象的「命名成就」具體化為疊高的實體星星，既考驗手部穩定度，又充滿遊戲樂趣！ 同時，B面「傾聽格」的設計非常巧妙，強迫孩子去注意「別人命名了什麼、為什麼」，直接落實了「傾聽與尊重」的社交核心能力！

🚀 進一步發展的方向

  1. 托盤疊星星的物理材質改良：星星配件如果較滑，放在棋子上的托盤移動時很容易翻覆，可能導致幼兒因過度挫折而氣餒。建議棋子托盤設計有淺凹槽，或星星配件採用有凹凸防滑刻紋的材質（例如木製梅花型積木或樂高積木），讓幼兒在移動棋子時能夠專注於走步。
  2. 發現卡與挑戰卡的平衡：挑戰卡的「失去星星」可能會讓好不容易疊高星星的幼兒大哭。建議將挑戰卡的負面效果改為「需暫停一次，由大家一起為他加油支持」，或「退後一步，但可移除一張限制卡」，以正向支持代替單純懲罰。
  3. 命名主題卡的引導：幼兒的字彙量有限，命名主題卡（如動物、食物）是非常好的輔助。可以額外提供一些「圖案提示卡」，當幼兒命名卡關時，可以抽一張圖案卡來發想。

🚀 進一步發展的方向

  1. 提高點讀筆的通用性與備用方案：點讀筆錄音貼紙在卡牌上如果磨損可能失效。建議點讀碼直接印刷在卡牌上，或者如果沒有點讀筆，也可以在說明書上印一張「QR code 音樂播放單」，讓老師用手機掃描直接播放對應音樂家的音樂，提高學具的通用性。
